[Deno](https://deno.land) is an open source [JavaScript](https://developer.mozilla.org/docs/Web/JavaScript)
runtime environment allowing the execution of JavaScript programs outside of the browser.

_Deno by Example_ is a hands-on introduction to Deno using annotated example programs. Check out the
[first example](examples/hello-world) or browse the full list below.
